ArrowAlert
==============

----
## What is ArrowAlert?
ArrowAlert is a HTML5 based mobile application that uses phonegap/cordova for native hooks.
The app uses an MVC design powered by angularjs, and a REST api to obtain data.

----
## Current Platforms:
* Android
* iOS
